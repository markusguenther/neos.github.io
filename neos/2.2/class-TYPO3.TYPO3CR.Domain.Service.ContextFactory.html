Class ContextFactory

The ContextFactory makes sure you don't create context instances with the same properties twice. Calling create() with the same parameters a second time will return the same Context instance again. Refer to 'ContextFactoryInterface' instead of 'ContextFactory' when injecting this factory into your own class.

TYPO3\TYPO3CR\Domain\Service\ContextFactory implements TYPO3\TYPO3CR\Domain\Service\ContextFactoryInterface
