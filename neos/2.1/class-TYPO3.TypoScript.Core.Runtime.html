TypoScript Runtime

TypoScript Rendering Process

During rendering, all TypoScript objects form a tree.

When a TypoScript object at a certain $typoScriptPath is invoked, it has access to all variables stored in the $context (which is an array).

The TypoScript object can then add or replace variables to this context using pushContext() or pushContextArray(), before rendering sub-TypoScript objects. After rendering these, it must call popContext() to reset the context to the last state.
