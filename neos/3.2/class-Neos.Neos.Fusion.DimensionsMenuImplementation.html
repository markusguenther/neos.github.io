Class DimensionsMenuImplementation

Fusion implementation for a dimensions menu.

The items generated by this menu will be all possible variants (according to the configured dimensions and presets) of the given node (including the given node).

If a 'dimension' is configured via Fusion, only the possible variants for that dimension will be included in the menu, any other dimensions will be kept from the current context.

Main Options: - dimension (optional, string): name of the dimension which this menu should be limited to. Example: "language". - presets (optional, array): If set, the presets are not loaded from the Settings, but instead taken from this property. Must be used with "dimension" set.
