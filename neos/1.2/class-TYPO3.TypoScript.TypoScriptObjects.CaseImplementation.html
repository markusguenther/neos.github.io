Class CaseImplementation

Case TypoScript Object

The "case" TypoScript object renders its children in order. The first result which is not MATCH_NORESULT is returned.

Often, this TypoScript object is used together with the "Matcher" TypoScript object; and all its children are by-default interpreted as "Matcher" TypoScript objects if no others are specified.
