# createImageVariant( array $processingInstructions, string $alias = null )

Create a variant of this image using the given processing instructions.

Create a variant of this image using the given processing instructions.

The variant is attached to the image for later (re-)use. If the optional alias parameter is specified, the image variant can later be retrieved via getImageVariantByAlias() An alias could, for example, be "thumbnail", "small", "micro", "face-emphasized" etc.

NOTE: If you want the new image variant to be persisted, make sure to update the image with ImageRepository::update()
