Class ImageViewHelper

Renders the src path of a thumbnail image of a given Neos.Media image instance

= Examples =

{neos.media:uri.image(image: imageObject)} (depending on the image) _Resources/Persistent/b29[...]95d.jpeg

{neos.media:uri.image(image: imageObject, maximumWidth: 80)} (depending on the image; has scaled keeping the aspect ratio) _Resources/Persistent/b29[...]95d.jpeg
