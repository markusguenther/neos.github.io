# pathinfo( string $path, integer $options = null )

Unicode variant of pathinfo() pathinfo() function is not unicode-friendly if setlocale is not set. It's sufficient to set it to any UTF-8 locale to correctly handle unicode strings. This wrapper function temporarily sets locale to 'en_US.UTF-8' and then restores original locale. It's not necessary to use this function in cases, where only file extension is determined, as it's hard to imagine a unicode file extension.
