Classes summary

Advisor

An advisor is the combination of a single advice and the pointcut where the advice will become active.
AspectContainer

An aspect is represented by class tagged with the "aspect" annotation. The aspect class may contain advices and pointcut declarations. Aspect classes are wrapped by this Aspect Container.
InterfaceIntroduction Implementation of the interface introduction declaration.
JoinPoint

In Flow the join point object contains context information when a point cut matches and the registered advices are executed.
PropertyIntroduction Implementation of the property introduction declaration.
TraitIntroduction Implementation of the trait introduction declaration.
