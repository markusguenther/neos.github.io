public boolean
# updateCredentials( TYPO3\Flow\Mvc\ActionRequest $actionRequest )

Updates the authentication credentials, the authentication manager needs to authenticate this token. This could be a username/password from a login controller. This method is called while initializing the security context. By returning TRUE you make sure that the authentication manager will (re-)authenticate the tokens with the current credentials. Note: You should not persist the credentials!

Updates the authentication credentials, the authentication manager needs to authenticate this token. This could be a username/password from a login controller.
