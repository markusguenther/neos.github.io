Class AjaxWidgetComponent

A HTTP component specifically for Ajax widgets It's task is to interrupt the default dispatching as soon as possible if the current request is an AJAX request triggered by a Fluid widget (e.g. contains the arguments "__widgetId" or "__widgetContext").
